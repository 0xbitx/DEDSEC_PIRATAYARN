<h1 align="center">PIRATAYARN</h1>
<h4 align="center">PIRATAYARN -  Video/Image/Music Downloader for Linux Users</h4>

## Description

**PIRATAYARN** is a lightweight, command-line downloader for Linux that fetches video, audio, and image content from over 50 online platforms. Designed for speed and simplicity, it requires no account and respects your freedom to save content locally.

Grab videos, music, reels, stories, GIFs, slideshows, and more—all free. Output high-quality MP4, MP3, JPG, and other formats directly to your system. No watermarks. No signups. No bloat. Just fast, reliable downloads from across the web.

Whether you're archiving a YouTube playlist, saving a TikTok clip, or pulling down a rare SoundCloud track—PIRATAYARN handles it cleanly and efficiently.

## Features

- **50+ Supported Sites** - YouTube, TikTok, SoundCloud, Spotify, Facebook, Instagram, and many more
- **No Watermarks** - Clean downloads without any branding or overlays
- **High Quality Downloads** - Preserve original quality in MP4, MP3, JPG, and other formats
- **Fast & Lightweight** - compressed code for maximum speed
- **Content Type Detection** - Auto-detects whether a link contains video, audio, or images
- **API Failover** - Automatically cycles through multiple API keys for reliability
- **Zero External Dependencies** - Runs entirely on Python's standard library
